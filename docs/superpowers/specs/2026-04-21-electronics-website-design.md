# 电子产品企业网站设计

## 概述

为华腾科技构建简约商务风格多页面网站，含首页、产品中心、关于我们、联系我们、管理后台。

## 技术栈

- **框架**：React + Vite + TypeScript
- **样式**：TailwindCSS（商务蓝白灰配色）
- **图片管理**：本地上传至 `public/uploads/` + URL 外部引用双轨并行
- **数据存储**：JSON 配置文件（`src/data/*.json`）
- **项目路径**：`/home/liuxue/桌面/electronics-website/`

## 页面结构

| 页面 | 路径 | 说明 |
|------|------|------|
| 首页 | `/` | 轮播图 + 核心优势 + 产品精选 |
| 产品中心 | `/products` | 产品列表网格，支持图片筛选 |
| 产品详情 | `/products/:id` | 产品图片画廊（主图+细节图）+ 详情描述 |
| 关于我们 | `/about` | 公司介绍、发展历程、团队 |
| 联系我们 | `/contact` | 留言表单 + 邮箱/电话/地址 |
| 管理后台 | `/admin` | 需登录，增删改产品/轮播图 |

## 首页设计

### 轮播图
- 3-5 张图片，自动轮播 + 手动切换
- 后台可配置图片 URL 与跳转链接
- 典型内容：新品发布、品牌形象、促销活动

### 核心优势
- 4 格横排展示：品质保证、技术创新、服务卓越、交付及时
- 图标 + 标题 + 简短描述

### 产品精选
- 6 格产品卡片（最新/精选）
- 悬停显示产品名与价格

## 产品中心

- 网格布局（PC 3-4 列，移动 1-2 列）
- 每卡片含：产品图、名称、简介
- 点击跳转详情页
- 图片来源：本地 `public/uploads/` 或外部 URL

## 产品详情页

- 左侧：图片画廊（缩略图列表 + 主图展示），支持多图
- 右侧：产品名称、型号、详细介绍、联系方式
- 多图支持主图+细节图切换

## 联系我们

- 左侧：联系信息（地址、电话、邮箱）
- 右侧：留言表单（姓名、邮箱、电话、内容）
- 表单提交存本地 JSON 或 localStorage

## 管理后台 `/admin`

### 登录
- 简单密码验证（硬编码或环境变量）
- 登录状态存 sessionStorage

### 产品管理
- 列表展示所有产品
- 新增：名称、描述、图片（上传/URL）
- 编辑：修改产品信息
- 删除：确认后删除

### 轮播图管理
- 列表展示轮播图
- 新增：图片（上传/URL）+ 跳转链接
- 启用/禁用切换
- 调整顺序

## 数据结构

```json
{
  "products": [
    {
      "id": "1",
      "name": "产品名",
      "description": "描述",
      "images": ["url1", "url2"],
      "featured": true,
      "createdAt": "2026-04-21"
    }
  ]
}
```

### 轮播图 `src/data/carousels.json`
```json
{
  "carousels": [
    {
      "id": "1",
      "image": "url",
      "link": "/products/1",
      "active": true
    }
  ]

```

```json
      "id": "1",

```

## 设计风格

- **主色调**：商务蓝 `#1E40AF` + 白色 + 浅灰 `#F3F4F6`
- **字体**：无衬线（系统默认或 Inter）
- **布局**：最大宽度 1280px，居中
- **间距**：统一 8px 基准网格
- **动效**：轻量过渡，悬停反馈，无过度动画

## 响应式断点

- `sm`: 640px
- `md`: 768px
- `lg`: 1024px
- `xl`: 1280px
